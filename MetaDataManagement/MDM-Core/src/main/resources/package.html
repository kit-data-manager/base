Core package for MetaDataManagement

This API implements an interface for persisting java instances. This module also contains all implementation for this interface. The configuration of the implemetations is done via a xml config file. The configuration contains:
  • implementation label
  • implementation class
  • supported 'persistence units'
The MetaDataManagement is implemented as a singleton and holds all implementations and its persistence units. The interface MetaDataManager provides all methods for persist instances. There should be only one instance of MetaDataManager for each 'persistence unit'. (recommended) The following class diagram shows the structure of this package:

Usage:

  1. get 'MetaDataManagement' (mdm = MetaDataManagement.getMetaDataManagement())
  2. get all possible implementations (List impls = mdm.getAllImplementations())
  3. get all persistence units of an implementation (units = mdm.getAllPersistenceUnits(impl.get(0))
  4. get 'MetaDataManager' (mdm.getMetaDataManager(impls.get(0), units.get(0))

Interface MetaDataManager

The interface supports the following tasks:
  • is instance already persisted
  • persist instance
  • update instance
  • find instance by primary key
  • find all instances of a given class
  • find all instances in a given range
